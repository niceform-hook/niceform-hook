# Hooks

O niceform-hook exporta todos os hooks do [react-hook-form](https://react-hook-form.com/docs/useNiceForm), 
otimizados para interagir perfeitamente com o contexto do niceform-hook. Portanto, para que funcionem corretamente, 
é necessário utilizar o provedor de contexto do seu formulário.
