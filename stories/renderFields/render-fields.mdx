# renderFields
`renderFields(data: Array<RenderFieldPayload | RenderFieldPayloadAsFunction>): React.ReactNode[]`

It works like [renderField](/docs/api-renderfield--docs), the difference is that the method receives as an argument an array containing the definitions of the fields you want to render.
