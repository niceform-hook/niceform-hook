# Niceform-hook

![imagem de exemplo](https://i.ibb.co/35DdCG40/niceform-hook.png)

Niceform-hook is a library written on top of react-hook-form, with its power you can easily render your form fields using json, saving time when writing code.

The niceform hook is very complete, designed precisely to meet different types of implementation and usage needs in a form.

<div className="subheading">Basic Example</div>

<Source
  language='jsx'
  dark
  format={true}
  code={`
import useNiceForm from './Form.config';\n

export function App(){
  const form = useNiceForm()

  return (
    <NiceformHookProvider form={form}>
        <form 
          className='grid grid-cols-4'
          onSubmit={form.methods.handleSubmit(console.log)}
        >
            <div>
              {form.renderField({
                type:'text',
                name: 'name',
                label: 'Name',
              })}
            </div>

            <div>
              {form.renderField({
                type: 'text',
                name: 'service_type',
                label: 'Service Type'
              })}
            </div>

            <div>
              {form.renderField({
                type: 'email',
                name: 'user',
                label: 'User'
              })}
            </div>
            
            <div>
              {form.renderField({
                type: 'password',
                name: 'password',
                label: 'Password'
              })}
            </div>

            <button type='submit'>Send</button>
        </form>
    </NiceformHookProvider>
  )
}
`}
/>
